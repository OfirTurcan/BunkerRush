# Bunker Rush - Setup Instructions

## Prerequisites
1. Install Node.js (v18+): https://nodejs.org/

## Installation & Run
```bash
cd C:\Code\Challenge
npm install
npm run dev
```
Then open http://localhost:5173 in your browser.

## Build for production
```bash
npm run build
npm run preview
```

## Controls
| Action | Keys |
|--------|------|
| Move Left | ← Arrow or A |
| Move Right | → Arrow or D |
| Jump | Space, W, or ↑ |
| Crouch | ↓ Arrow or S |
| Mobile | On-screen buttons |
